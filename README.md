# Unscroll

> 스크롤을 줄이고, 원하는 정보에 더 빨리 도달하게 만드는 **콘텐츠 정리/요약(또는 필터링) 기반 도구**입니다.  
> (한 줄 설명은 프로젝트 성격에 맞게 수정하세요)

---

## 한눈에 보기

- **문제의식**: 끝없이 스크롤하며 시간을 소모하는 사용 경험(피로도)을 줄이고 싶었습니다.
- **해결 아이디어**: 핵심 정보만 남기고 불필요한 요소를 덜어내어, 빠르게 맥락을 파악할 수 있게 합니다.
- **내가 한 일**: 기능 설계, 구현, 테스트 및 사용성 개선을 수행했습니다.

## 주요 기능

- 콘텐츠에서 **핵심만 추려보기** (예: 요약/하이라이트/중요 문장 추출)
- **불필요한 요소 숨김** (예: 반복/광고/잡음 제거 등 프로젝트에 맞게 수정)
- 사용자 설정 기반 **표시 방식 커스터마이징**
- (선택) 기록/북마크/내보내기 기능

> 위 항목은 실제 구현된 기능에 맞춰 남기고 삭제/수정하세요.
