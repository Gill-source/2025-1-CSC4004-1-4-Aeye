## 👀 [ About 'Unscorll' ]
Unscroll은 **스크롤없이** 한 눈에 요약된 쇼핑 정보를 얻을 수 있는 우리 곁의 가장 가까운 퍼스널 AI 쇼핑 비서입니다.

- **문제의식**: 정보 접근이 점점 “긴 스크롤 + 복잡한 UI” 중심으로 설계되면서, 고령층·장애인·디지털 문해력이 낮은 사용자 등 **디지털 취약계층**이 필요한 정보를 찾기까지 더 많은 시간과 노력을 요구받고 있습니다. Unscroll은 불필요한 탐색 부담을 줄이고 핵심 정보에 더 빠르게 도달할 수 있도록 돕는 것을 목표로 합니다.
- **해결 아이디어**: 핵심 정보만 남기고 불필요한 요소를 덜어내어, 빠르게 맥락을 파악할 수 있게 합니다.

## 주요 기능

- 콘텐츠에서 **핵심만 추려보기** (예: 요약/하이라이트/중요 문장 추출)
